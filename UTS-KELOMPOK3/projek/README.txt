Folder Projek Keseluruhan berada di dalam folder dibanten_frontend.

Untuk dapat menjalankannya silahkan hidupkan XAMPP (Apache dan MySQL). Terdapat dua role didalamnya yaitu pembeli dan penjual. 

--> URL halaman Pembeli
http://localhost/dibanten_frontend/pembeli/content/index.php

--> URL halaman Penjual
http://localhost/dibanten_frontend/penjual/content/index.php


=======
WARNING
=======
Folder dengan nama "templates" bukan template tampilan yang didownload dari internet. Melainkan hanya nama folder saja yang berjudul "templates". Di dalamnya merupakan header dan footer dari file html agar tidak menduplikasi penulisannya di setiap page yang ada sehingga lebih efisien dan efektif (teknik bootstraping).
